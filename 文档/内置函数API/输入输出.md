# 输入输出

## 打印

输出内容到控制台。

**签名**：`打印(*值, 分隔符: 文本 = " ", 结尾: 文本 = "\n")`

**参数**：
- `*值`：要输出的值，可传入多个
- `分隔符`：多个值之间的分隔符，默认为空格
- `结尾`：输出末尾的字符，默认为换行符

**返回**：`空`

**说明**：
- 布尔值 `真`/`假` 输出为中文
- `空` 值输出为"空"
- 对象实例输出其字符串表示

**示例**：
```
打印("你好，世界！")  // 你好，世界！
打印(1, 2, 3)  // 1 2 3
打印(1, 2, 3, 分隔符=", ")  // 1, 2, 3
打印("不换行", 结尾="")  // 输出后不换行
打印(真)  // 真
打印(假)  // 假
打印(空)  // 空
```

## 输入

从控制台读取用户输入。

**签名**：`输入(提示: 文本 = "")`

**参数**：
- `提示`：输入提示文本

**返回**：用户输入的字符串

**示例**：
```
定义 名字 = 输入("请输入你的名字：")
打印("你好，" + 名字)
```
