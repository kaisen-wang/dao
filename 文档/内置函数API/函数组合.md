# 函数组合

## 柯里化

将多参数函数转换为单参数函数链。

**签名**：`柯里化(函数: 函数) -> 函数`

**异常**：`类型错误`：参数不是函数

**示例**：
```
定义 加 = 函数(a, b) => a + b
定义 加5 = 柯里化(加)(5)
打印(加5(3))  // 8
打印(加5(10)) // 15
```

---

## 组合

从右到左组合多个函数（传统数学定义）。

**签名**：`组合(*函数) -> 函数`

**说明**：`组合(f, g)` 返回 `x => f(g(x))`

**异常**：
- `运行时错误`：未提供任何函数
- `类型错误`：参数不是函数

**示例**：
```
定义 双倍 = 函数(x) => x * 2
定义 加一 = 函数(x) => x + 1

定义 双倍后加一 = 组合(加一, 双倍)
打印(双倍后加一(3))  // 7 （3 * 2 + 1）

定义 加一后双倍 = 组合(双倍, 加一)
打印(加一后双倍(3))  // 8 （(3 + 1) * 2）
```

## 管道组合

从左到右组合多个函数（管道风格）。

**签名**：`管道组合(*函数) -> 函数`

**说明**：`管道组合(f, g, h)` 等价于 `x => h(g(f(x)))`

**异常**：
- `运行时错误`：未提供任何函数
- `类型错误`：参数不是函数

**示例**：
```
定义 双倍 = 函数(x) => x * 2
定义 加一 = 函数(x) => x + 1

定义 处理 = 管道组合(双倍, 加一)
打印(处理(3))  // 7 （3 * 2 + 1，先双倍再加一）
```
