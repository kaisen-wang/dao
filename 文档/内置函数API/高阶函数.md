# 高阶函数

高阶内置函数接受函数作为参数，用于函数式编程。这些函数无需导入，可直接使用。

## 映射

对集合中的每个元素应用函数，返回新列表。

**签名**：`映射(集合: 列表, 函数: 函数) -> 列表`

**参数**：
- `集合`：要处理的列表
- `函数`：接受一个参数的函数

**返回**：映射后的新列表

**示例**：
```
定义 结果 = 映射([1, 2, 3], 函数(x) => x * 2)
打印(结果)  // [2, 4, 6]
```

---

## 筛选

保留集合中满足条件的元素。

**签名**：`筛选(集合: 列表, 函数: 函数) -> 列表`

**参数**：
- `集合`：要筛选的列表
- `函数`：接受一个参数、返回布尔值的函数

**返回**：筛选后的新列表

**示例**：
```
定义 结果 = 筛选([1, 2, 3, 4, 5], 函数(x) => x > 3)
打印(结果)  // [4, 5]
```

---

## 折叠

从初始值开始，逐个合并集合中的元素。

**签名**：`折叠(集合: 列表, 初始值: 任意, 函数: 函数) -> 任意`

**参数**：
- `集合`：要折叠的列表
- `初始值`：累积的初始值
- `函数`：接受两个参数（累积值、当前元素）的函数

**返回**：折叠后的最终结果

**示例**：
```
定义 求和 = 折叠([1, 2, 3, 4, 5], 0, 函数(累加, 当前) => 累加 + 当前)
打印(求和)  // 15

定义 拼接 = 折叠(["a", "b", "c"], "", 函数(累加, 当前) => 累加 + 当前)
打印(拼接)  // "abc"
```

## 每个满足

检查集合中所有元素是否都满足条件。

**签名**：`每个满足(集合: 列表, 函数: 函数) -> 布尔`

**示例**：
```
打印(每个满足([2, 4, 6], 函数(x) => x % 2 == 0))  // 真
打印(每个满足([2, 3, 6], 函数(x) => x % 2 == 0))  // 假
```

---

## 存在满足

检查集合中是否有元素满足条件。

**签名**：`存在满足(集合: 列表, 函数: 函数) -> 布尔`

**示例**：
```
打印(存在满足([1, 2, 3], 函数(x) => x > 2))  // 真
打印(存在满足([1, 2, 3], 函数(x) => x > 5))  // 假
```

---

## 查找

找到集合中第一个满足条件的元素。

**签名**：`查找(集合: 列表, 函数: 函数) -> 任意`

**返回**：第一个满足条件的元素，未找到返回 `空`

**示例**：
```
打印(查找([1, 2, 3, 4], 函数(x) => x > 2))  // 3
打印(查找([1, 2, 3], 函数(x) => x > 5))     // 空
```

---

## 排序

对集合进行排序。

**签名**：`排序(集合: 列表, 键: 函数 = 空, 降序: 布尔 = 假) -> 列表`

**参数**：
- `集合`：要排序的列表
- `键`：排序键函数，省略则按自然顺序
- `降序`：是否降序排列

**返回**：排序后的新列表

**示例**：
```
打印(排序([3, 1, 4, 1, 5]))  // [1, 1, 3, 4, 5]
打印(排序([3, 1, 4, 1, 5], 降序=真))  // [5, 4, 3, 1, 1]
打印(排序(["banana", "apple", "cherry"], 键=函数(s) => 长度(s)))  // ["apple", "banana", "cherry"]
```

## 展平映射

对集合中的每个元素应用函数，然后将结果展平一层。

**签名**：`展平映射(集合: 列表, 函数: 函数) -> 列表`

**示例**：
```
定义 结果 = 展平映射([1, 2, 3], 函数(x) => [x, x * 2])
打印(结果)  // [1, 2, 2, 4, 3, 6]
```

---

## 分组

按函数返回值将集合元素分组。

**签名**：`分组(集合: 列表, 函数: 函数) -> 字典`

**返回**：键为分组标识、值为元素列表的字典

**示例**：
```
定义 结果 = 分组([1, 2, 3, 4, 5, 6], 函数(x) => x % 2)
// {0: [2, 4, 6], 1: [1, 3, 5]}
```

---

## 映射流

对流进行惰性映射，返回新流。

**签名**：`映射流(流: 流, 函数: 函数) -> 流`

**说明**：惰性求值，仅在消费流时才计算。

**示例**：
```
定义 s = 映射流([1, 2, 3], 函数(x) => x * 2)
定义 结果 = 取(s, 2)
打印(结果)  // [2, 4]
```

---

## 筛选流

对流进行惰性筛选，返回新流。

**签名**：`筛选流(流: 流, 函数: 函数) -> 流`

**示例**：
```
定义 s = 筛选流([1, 2, 3, 4, 5], 函数(x) => x > 2)
定义 结果 = 取(s, 2)
打印(结果)  // [3, 4]
```

---

## 取

从流中取前 n 个值，返回列表。

**签名**：`取(流: 流, n: 整数) -> 列表`

**示例**：
```
定义 s = 映射流(范围(100), 函数(x) => x * x)
打印(取(s, 5))  // [0, 1, 4, 9, 16]
```
