# 集合操作

## 范围

生成整数范围列表。

**签名**：
- `范围(结束: 整数) -> 列表` — 生成 0 到结束的列表
- `范围(开始: 整数, 结束: 整数) -> 列表` — 生成开始到结束的列表（含两端）
- `范围(开始: 整数, 结束: 整数, 步长: 整数) -> 列表` — 按步长生成

**异常**：`运行时错误`：参数数量不正确

**示例**：
```
打印(范围(5))         // [0, 1, 2, 3, 4]
打印(范围(1, 5))      // [1, 2, 3, 4, 5]
打印(范围(0, 10, 2))  // [0, 2, 4, 6, 8, 10]
```

---

## 追加

向列表末尾追加元素（修改原列表）。

**签名**：`追加(列表: 列表, 元素: 任意) -> 列表`

**异常**：`类型错误`：第一个参数不是列表

**示例**：
```
定义 列 = [1, 2, 3]
追加(列, 4)
打印(列)  // [1, 2, 3, 4]
```

---

## 反转

反转列表或字符串，返回新对象（不修改原对象）。

**签名**：`反转(对象: 列表|文本) -> 列表|文本`

**异常**：`类型错误`：类型不支持反转

**示例**：
```
打印(反转([1, 2, 3]))   // [3, 2, 1]
打印(反转("你好"))      // "好你"
```

---

## 包含

检查集合是否包含指定元素。

**签名**：`包含(集合: 任意, 元素: 任意) -> 布尔`

**示例**：
```
打印(包含([1, 2, 3], 2))    // 真
打印(包含("你好世界", "世界"))  // 真
打印(包含({"a": 1}, "a"))   // 真
```

---

## 集合

创建集合。

**签名**：
- `集合() -> 集合` — 创建空集合
- `集合(可迭代对象) -> 集合` — 从可迭代对象创建集合
- `集合(元素1, 元素2, ...) -> 集合` — 从多个元素创建集合

**示例**：
```
打印(集合())              // set()
打印(集合([1, 2, 2, 3]))  // {1, 2, 3}
打印(集合(1, 2, 3))       // {1, 2, 3}
```

---

## 交集

计算两个集合的交集。

**签名**：`交集(集合A: 集合, 集合B: 集合) -> 集合`

**示例**：
```
打印(交集({1, 2, 3}, {2, 3, 4}))  // {2, 3}
```

---

## 并集

计算两个集合的并集。

**签名**：`并集(集合A: 集合, 集合B: 集合) -> 集合`

**示例**：
```
打印(并集({1, 2}, {3, 4}))  // {1, 2, 3, 4}
```
