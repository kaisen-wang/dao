# 数学运算

## 最大值

求最大值。

**签名**：
- `最大值(可迭代对象) -> 任意` — 求可迭代对象中的最大值
- `最大值(值1, 值2, ...) -> 任意` — 求多个值中的最大值

**示例**：
```
打印(最大值(3, 1, 4, 1, 5))   // 5
打印(最大值([3, 1, 4, 1, 5]))  // 5
```

## 绝对值

求绝对值。

**签名**：`绝对值(x: 数值) -> 数值`

**示例**：
```
打印(绝对值(-5))    // 5
打印(绝对值(3.14))  // 3.14
```
