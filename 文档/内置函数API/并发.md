# 并发

## 睡眠

暂停当前执行指定的秒数。

**签名**：`睡眠(秒数: 小数) -> 空`

**示例**：
```
睡眠(1)      // 暂停1秒
睡眠(0.5)    // 暂停0.5秒
```

**方法**：

| 方法名 | 说明 |
|--------|------|
| `获取()` | 获取锁 |
| `释放()` | 释放锁 |

**示例**：
```
定义 锁 = 互斥锁()
锁.获取()
// 临界区代码
锁.释放()
```

## 原子整数

创建一个线程安全的原子整数。

**签名**：`原子整数(初始值: 整数 = 0) -> AtomicInt`

**方法**：

| 方法名 | 说明 |
|--------|------|
| `获取()` | 获取当前值 |
| `设置(值)` | 设置新值 |
| `加并获取(增量)` | 原子加法并返回新值 |
| `比较并设置(期望, 新值)` | CAS 操作 |

**示例**：
```
定义 计数 = 原子整数(0)
计数.加并获取(1)
打印(计数.获取())  // 1
```

## 运行异步

运行异步函数并等待结果。

**签名**：`运行异步(函数: 异步函数|协程) -> 任意`

**参数**：
- `函数`：异步函数、协程对象或可调用对象

**异常**：`类型错误`：参数不是异步函数或协程

**示例**：
```
异步 函数 获取数据()
    返回 "数据"

定义 结果 = 运行异步(获取数据)
打印(结果)  // "数据"
```
