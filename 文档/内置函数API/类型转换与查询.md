# 类型转换与查询

## 取类型

获取值的类型名称。

**签名**：`取类型(值: 任意) -> 文本`

**返回**：类型名称字符串

**类型名称映射**：

| 道语言类型 | 返回值 |
|-----------|--------|
| 整数/浮点数 | `"数值"` |
| 字符串 | `"文本"` |
| 布尔值 | `"布尔"` |
| 列表 | `"列表"` |
| 字典 | `"字典"` |
| 元组 | `"元组"` |
| 集合 | `"集合"` |
| 空 | `"空"` |
| 函数 | `"函数"` |
| 类实例 | 类名 |
| 类 | `"类型"` |

**示例**：
```
打印(取类型(42))       // "数值"
打印(取类型("你好"))    // "文本"
打印(取类型(真))       // "布尔"
打印(取类型([1,2,3]))  // "列表"
打印(取类型(空))       // "空"
```

---

## 整数

将值转换为整数。

**签名**：`整数(值: 任意) -> 整数`

**异常**：`类型错误`：无法转换为整数

**示例**：
```
打印(整数("42"))    // 42
打印(整数(3.7))     // 3
打印(整数(真))      // 1
```

---

## 小数

将值转换为浮点数。

**签名**：`小数(值: 任意) -> 小数`

**异常**：`类型错误`：无法转换为小数

**示例**：
```
打印(小数("3.14"))  // 3.14
打印(小数(42))      // 42.0
```

---

## 文本

将值转换为字符串。

**签名**：`文本(值: 任意) -> 文本`

**说明**：
- 布尔值 `真` 转为 `"真"`，`假` 转为 `"假"`
- `空` 值转为 `"空"`

**示例**：
```
打印(文本(42))    // "42"
打印(文本(真))    // "真"
打印(文本(空))    // "空"
```

---

## 长度

获取集合或字符串的长度。

**签名**：`长度(对象: 任意) -> 整数`

**参数**：
- `对象`：字符串、列表、字典、元组或集合

**异常**：`类型错误`：类型不支持求长度操作

**示例**：
```
打印(长度("你好"))     // 2
打印(长度([1,2,3]))   // 3
打印(长度({"a":1}))   // 1
```

---

## 是实例

检查对象是否是某个类型的实例（含继承关系）。

**签名**：`是实例(对象: 任意, 类型: 类型) -> 布尔`

**参数**：
- `对象`：要检查的对象
- `类型`：目标类型（必须是道语言的类）

**异常**：`类型错误`：第二个参数必须是类型

**示例**：
```
定义 动物 = 类("动物")
定义 狗 = 类("狗", 继承自=动物)

定义 d = 狗()
打印(是实例(d, 狗))    // 真
打印(是实例(d, 动物))  // 真
```
