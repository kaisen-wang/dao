# 道语言内置函数 API 总览

道语言内置函数是无需导入即可在全局环境中直接使用的函数。分为两类：

1. **基础内置函数**：纯 Python 函数，无需解释器引用
2. **高阶内置函数**：需要解释器引用，用于函数式编程
