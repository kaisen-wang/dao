# 逻辑编程

## 添加事实

向知识库动态添加事实。

**签名**：`添加事实(知识库: 知识库, 谓词: 文本, *参数: 任意) -> 空`

**参数**：
- `知识库`：知识库对象
- `谓词`：事实的谓词名称
- `*参数`：事实的参数

**异常**：`类型错误`：第一个参数不是知识库

**示例**：
```
逻辑块 动物知识库
    事实 哺乳动物(猫)
    事实 哺乳动物(狗)
结束

添加事实(动物知识库, "哺乳动物", "兔")
```

## 删除事实

从知识库动态删除事实。

**签名**：`删除事实(知识库: 知识库, 谓词: 文本, *参数: 任意) -> 空`

**参数**：
- `知识库`：知识库对象
- `谓词`：事实的谓词名称
- `*参数`：事实的参数

**异常**：`类型错误`：第一个参数不是知识库

**示例**：
```
删除事实(动物知识库, "哺乳动物", "兔")
```
