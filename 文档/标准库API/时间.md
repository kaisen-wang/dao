# 时间模块 API

模块路径：`时间`

## 导入方式

```
从 时间 导入 现在, 解析, 格式化
从 时间 导入 计时器, 睡眠
导入 时间
```

## 函数列表

### 现在

获取当前时间，返回 `DaoDateTime` 对象。

**签名**：`现在() -> DaoDateTime`

**示例**：
```
定义 当前 = 现在()
打印(当前.年份)  // 2026
打印(当前.月份)  // 6
```

---

### 解析

将时间字符串解析为 `DaoDateTime` 对象。

**签名**：`解析(文本: 文本, 格式: 文本) -> DaoDateTime`

**参数**：
- `文本`：时间字符串
- `格式`：格式字符串，使用道语言占位符

**格式占位符**：

| 占位符 | 说明 | 示例 |
|--------|------|------|
| `YYYY` | 四位年份 | 2026 |
| `MM` | 两位月份 | 02 |
| `DD` | 两位日期 | 09 |
| `HH` | 两位小时 | 15 |
| `mm` | 两位分钟 | 30 |
| `ss` | 两位秒 | 00 |

**异常**：`运行时错误`：无法解析时间字符串

**示例**：
```
定义 生日 = 解析("2000-01-15", "YYYY-MM-DD")
打印(生日.年份)  // 2000
```

---

### 格式化

将 `DaoDateTime` 对象格式化为字符串。

**签名**：`格式化(时间: DaoDateTime, 格式: 文本) -> 文本`

**参数**：
- `时间`：`DaoDateTime` 对象
- `格式`：格式字符串，使用道语言占位符（同 `解析`）

**异常**：`运行时错误`：参数类型不正确

**示例**：
```
定义 当前 = 现在()
打印(格式化(当前, "YYYY年MM月DD日"))  // "2026年06月29日"
```

---

### 计时器

创建并返回一个 `DaoTimer` 计时器对象。

**签名**：`计时器() -> DaoTimer`

**示例**：
```
定义 计时 = 计时器()
计时.开始()
// ... 执行一些操作 ...
打印("耗时: {计时.停止()} 毫秒")
```

---

### 睡眠

暂停当前执行指定的秒数。

**签名**：`睡眠(秒数: 小数) -> 空`

**参数**：
- `秒数`：暂停的秒数

**示例**：
```
睡眠(1)  // 暂停1秒
```

---

## 自定义类型

### DaoDateTime

时间日期对象，包装 Python `datetime.datetime`。

**属性**：

| 属性名 | 类型 | 说明 |
|--------|------|------|
| `年份` | 整数 | 年 |
| `月份` | 整数 | 月（1-12） |
| `日` | 整数 | 日（1-31） |
| `小时` | 整数 | 时（0-23） |
| `分钟` | 整数 | 分（0-59） |
| `秒` | 整数 | 秒（0-59） |
| `微秒` | 整数 | 微秒 |

**运算**：

| 运算符 | 说明 | 返回类型 |
|--------|------|---------|
| `a - b` | 计算两个时间的差 | `DaoTimeDelta` |

---

### DaoTimeDelta

时间差对象，包装 Python `datetime.timedelta`。

**属性**：

| 属性名 | 类型 | 说明 |
|--------|------|------|
| `总秒数` | 小数 | 时间差的总秒数 |
| `天数` | 整数 | 时间差的天数 |

**示例**：
```
定义 dt1 = 解析("2026-06-01", "YYYY-MM-DD")
定义 dt2 = 解析("2026-05-01", "YYYY-MM-DD")
定义 差 = dt1 - dt2
打印(差.天数)  // 31
```

---

### DaoTimer

计时器对象，基于高精度计时器。

**方法**：

| 方法名 | 签名 | 说明 |
|--------|------|------|
| `开始` | `开始() -> 空` | 开始计时 |
| `停止` | `停止() -> 小数` | 停止计时，返回经过的毫秒数 |

**异常**：`运行时错误`：在未开始计时时调用 `停止`
