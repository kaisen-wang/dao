# 集合模块 API

模块路径：`集合`

## 导入方式

```
从 集合 导入 创建堆, 入堆, 出堆
从 集合 导入 有序字典, 双端队列
导入 集合
```

## 函数列表

### 创建堆

创建一个空的最小堆。

**签名**：`创建堆() -> DaoHeap`

**示例**：
```
定义 h = 创建堆()
入堆(h, 3)
入堆(h, 1)
入堆(h, 2)
打印(出堆(h))  // 1
```

---

### 入堆

向堆中插入元素。

**签名**：`入堆(堆: DaoHeap, 元素: 任意) -> 空`

**参数**：
- `堆`：`DaoHeap` 对象
- `元素`：要插入的元素

**异常**：`运行时错误`：参数不是堆类型

---

### 出堆

从堆中弹出最小元素。

**签名**：`出堆(堆: DaoHeap) -> 任意`

**异常**：
- `运行时错误`：参数不是堆类型
- `索引错误`：堆为空

---

### 计数

统计列表中各元素出现的次数。

**签名**：`计数(列表: 列表) -> DaoCounter`

**示例**：
```
定义 c = 计数(["甲", "乙", "甲", "丙", "甲"])
打印(c.取值("甲"))  // 3
```

---

### 最常见

返回计数器中出现次数最多的元素。

**签名**：`最常见(计数器: DaoCounter, n: 整数 = 空) -> 列表`

**参数**：
- `计数器`：`DaoCounter` 对象
- `n`：返回前 n 个，省略则返回全部

**异常**：`运行时错误`：参数不是计数器类型

**示例**：
```
定义 c = 计数(["甲", "乙", "甲", "丙", "甲"])
打印(最常见(c, 2))  // [("甲", 3), ("乙", 1)]
```

---

## 自定义类型

### DaoHeap

最小堆数据结构。

**方法**：

| 方法名 | 签名 | 说明 |
|--------|------|------|
| `push` | `push(元素) -> 空` | 插入元素 |
| `pop` | `pop() -> 任意` | 弹出最小元素 |
| `peek` | `peek() -> 任意` | 查看最小元素（不弹出） |
| `is_empty` | `is_empty() -> 布尔` | 堆是否为空 |
| `size` | `size() -> 整数` | 堆中元素数量 |

**异常**：
- `索引错误`：空堆调用 `pop` 或 `peek`

---

### DaoCounter

计数器，统计元素出现次数。

**方法**：

| 方法名 | 签名 | 说明 |
|--------|------|------|
| `取值` | `取值(键, 默认值=0) -> 整数` | 获取指定键的计数 |
| `取键列表` | `取键列表() -> 列表` | 获取所有键 |
| `最常见` | `最常见(n=空) -> 列表` | 返回出现次数最多的前 n 个元素 |

---

### DaoOrderedDict

有序字典，保持插入顺序。

**方法**：

| 方法名 | 签名 | 说明 |
|--------|------|------|
| `设置` | `设置(键, 值) -> 空` | 设置键值对 |
| `取值` | `取值(键, 默认值=空) -> 任意` | 获取值 |
| `取键列表` | `取键列表() -> 列表` | 获取所有键（按插入顺序） |
| `取值列表` | `取值列表() -> 列表` | 获取所有值（按插入顺序） |
| `删除` | `删除(键) -> 空` | 删除键值对 |

**异常**：`运行时错误`：删除不存在的键

---

### DaoDeque

双端队列。

**方法**：

| 方法名 | 签名 | 说明 |
|--------|------|------|
| `左入` | `左入(元素) -> 空` | 左端插入 |
| `右入` | `右入(元素) -> 空` | 右端插入 |
| `左出` | `左出() -> 任意` | 左端弹出 |
| `右出` | `右出() -> 任意` | 右端弹出 |
| `左窥` | `左窥() -> 任意` | 查看左端元素 |
| `右窥` | `右窥() -> 任意` | 查看右端元素 |
| `size` | `size() -> 整数` | 元素数量 |
| `is_empty` | `is_empty() -> 布尔` | 是否为空 |

**异常**：`索引错误`：空队列调用弹出或查看方法

---

### DaoDefaultDict

默认字典，访问不存在的键时返回默认值。

**方法**：

| 方法名 | 签名 | 说明 |
|--------|------|------|
| `取值` | `取值(键) -> 任意` | 获取值，不存在则返回默认值 |
| `设置` | `设置(键, 值) -> 空` | 设置键值对 |
| `取键列表` | `取键列表() -> 列表` | 获取所有键 |

---

### DaoFrozenSet

冻结集合（不可变集合）。

**方法**：

| 方法名 | 签名 | 说明 |
|--------|------|------|
| `包含` | `包含(元素) -> 布尔` | 检查是否包含元素 |
| `size` | `size() -> 整数` | 元素数量 |
