# 日志模块 API

模块路径：`日志`

## 导入方式

```
从 日志 导入 创建日志器, 创建文件处理器, 创建控制台处理器
导入 日志
```

## 函数列表

### 创建日志器

创建一个日志器。

**签名**：`创建日志器(名称: 文本) -> DaoLogger`

**参数**：
- `名称`：日志器名称

**示例**：
```
定义 日志 = 创建日志器("我的应用")
```

---

### 创建文件处理器

创建一个文件日志处理器。

**签名**：`创建文件处理器(路径: 文本) -> DaoFileHandler`

**参数**：
- `路径`：日志文件路径

**异常**：`运行时错误`：无法创建日志文件

**示例**：
```
定义 文件处理器 = 创建文件处理器("app.log")
```

---

### 创建控制台处理器

创建一个控制台日志处理器。

**签名**：`创建控制台处理器() -> DaoConsoleHandler`

**示例**：
```
定义 控制台处理器 = 创建控制台处理器()
```

---

## 自定义类型

### DaoLogger

日志器对象。

**方法**：

| 方法名 | 签名 | 说明 |
|--------|------|------|
| `调试` | `调试(消息: 文本) -> 空` | 输出调试级别日志 |
| `信息` | `信息(消息: 文本) -> 空` | 输出信息级别日志 |
| `警告` | `警告(消息: 文本) -> 空` | 输出警告级别日志 |
| `错误` | `错误(消息: 文本) -> 空` | 输出错误级别日志 |
| `设置级别` | `设置级别(级别: 文本) -> 空` | 设置日志级别 |
| `添加处理器` | `添加处理器(处理器) -> 空` | 添加输出处理器 |
| `设置格式` | `设置格式(格式: 文本) -> 空` | 设置输出格式 |

**日志级别**：

| 道语言级别 | 说明 | 数值 |
|-----------|------|------|
| `调试` | 调试信息 | 10 |
| `信息` | 一般信息 | 20 |
| `警告` | 警告信息 | 30 |
| `错误` | 错误信息 | 40 |

**格式占位符**：

| 占位符 | 说明 |
|--------|------|
| `{级别}` | 日志级别 |
| `{时间}` | 时间戳 |
| `{名称}` | 日志器名称 |
| `{消息}` | 日志消息 |

**异常**：`运行时错误`：设置无效的日志级别

**示例**：
```
定义 日志 = 创建日志器("应用")
日志.添加处理器(创建控制台处理器())
日志.设置级别("信息")
日志.设置格式("[{级别}] {时间} - {消息}")
日志.信息("程序启动")
日志.警告("磁盘空间不足")
日志.错误("连接失败")
```

---

### DaoFileHandler

文件日志处理器。

**方法**：

| 方法名 | 签名 | 说明 |
|--------|------|------|
| `关闭` | `关闭() -> 空` | 关闭文件处理器 |

---

### DaoConsoleHandler

控制台日志处理器。无额外方法。
