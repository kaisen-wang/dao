## 函数列表

### 三角函数

#### 正弦

计算正弦值。

**签名**：`正弦(x: 小数) -> 小数`

**示例**：`正弦(0)` → `0.0`

#### 余弦

计算余弦值。

**签名**：`余弦(x: 小数) -> 小数`

**示例**：`余弦(0)` → `1.0`

#### 正切

计算正切值。

**签名**：`正切(x: 小数) -> 小数`

**示例**：`正切(0)` → `0.0`

#### 反正弦

计算反正弦值。

**签名**：`反正弦(x: 小数) -> 小数`

**异常**：`运行时错误`：参数超出定义域（|x| > 1）

**示例**：`反正弦(1)` → 约 `1.5708`（π/2）

#### 反余弦

计算反余弦值。

**签名**：`反余弦(x: 小数) -> 小数`

**异常**：`运行时错误`：参数超出定义域（|x| > 1）

#### 反正切

计算反正切值。

**签名**：`反正切(x: 小数) -> 小数`

### 随机数函数

#### 随机整数

生成指定范围内的随机整数。

**签名**：`随机整数(最小: 整数, 最大: 整数) -> 整数`

**参数**：
- `最小`：最小值（含）
- `最大`：最大值（含）

**异常**：`运行时错误`：最小值大于最大值

**示例**：`随机整数(1, 10)` → 1到10之间的整数

#### 随机小数

生成 0.0 到 1.0 之间的随机浮点数。

**签名**：`随机小数() -> 小数`

**示例**：`随机小数()` → 0.0 到 1.0 之间的浮点数

#### 随机选择

从列表中随机选择一个元素。

**签名**：`随机选择(列表: 列表) -> 任意`

**异常**：`运行时错误`：列表为空

**示例**：`随机选择(["甲", "乙", "丙"])` → 列表中的一个元素

#### 打乱

随机打乱列表顺序，返回新列表（不修改原列表）。

**签名**：`打乱(列表: 列表) -> 列表`

**示例**：`打乱([1, 2, 3, 4, 5])` → 原列表的随机排列

### 统计函数

#### 平均值

计算列表的平均值。

**签名**：`平均值(列表: 列表) -> 小数`

**异常**：`运行时错误`：列表为空

**示例**：`平均值([1, 2, 3, 4, 5])` → `3.0`

#### 中位数

计算列表的中位数。

**签名**：`中位数(列表: 列表) -> 小数`

**异常**：`运行时错误`：列表为空

**示例**：`中位数([1, 2, 3, 4, 5])` → `3`

#### 标准差

计算列表的标准差。

**签名**：`标准差(列表: 列表) -> 小数`

**异常**：`运行时错误`：列表元素少于2个

**示例**：`标准差([2, 4, 4, 4, 5, 5, 7, 9])` → 约 `2.14`

#### 向下取整

#### 四舍五入

四舍五入取整。

### 幂和对数函数

#### 幂

计算幂运算。

**签名**：`幂(底: 小数, 指: 小数) -> 小数`

**示例**：`幂(2, 10)` → `1024.0`

#### 自然对数

计算自然对数。

**签名**：`自然对数(x: 小数) -> 小数`

**异常**：`运行时错误`：参数必须为正数

**示例**：`自然对数(数学.自然常数)` → 约 `1.0`

#### 对数

计算指定底数的对数。

**签名**：`对数(x: 小数, 底: 小数) -> 小数`

**异常**：`运行时错误`：参数无效

**示例**：`对数(100, 10)` → `2.0`
