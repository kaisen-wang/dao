# 道语言标准库 API 总览

道语言标准库提供 10 个模块，涵盖日常开发所需的核心功能。所有 API 均使用中文命名，遵循统一的命名规范。

## 模块列表

| 模块 | 路径 | 说明 | 详细文档 |
|------|------|------|---------|
| 文本 | `文本` | 字符串处理：分割、连接、替换、正则匹配等 | [文本.md](文本.md) |
| 数学 | `数学` | 数学计算：三角函数、随机数、统计、取整、幂对数 | [数学.md](数学.md) |
| 时间 | `时间` | 日期与时间：解析、格式化、计时器、睡眠 | [时间.md](时间.md) |
| 集合 | `集合` | 数据结构：堆、计数器、有序字典、双端队列等 | [集合.md](集合.md) |
| 编码 | `编码` | 数据编码：JSON、CSV、Base64、URL | [编码.md](编码.md) |
| 文件 | `文件` | 文件系统：读写、遍历、复制、移动、删除 | [文件.md](文件.md) |
| 系统 | `系统` | 系统交互：环境变量、命令行、进程、目录 | [系统.md](系统.md) |
| 测试 | `测试` | 测试框架：断言、测试套件、测试报告 | [测试.md](测试.md) |
| 日志 | `日志` | 日志记录：分级日志、文件/控制台输出 | [日志.md](日志.md) |
| 网络 | `网络` | 网络通信：HTTP 请求/服务、URL 编解码 | [网络.md](网络.md) |

## 导入语法

```
// 导入指定函数
从 文本 导入 分割, 连接, 替换

// 导入整个模块
导入 数学
数学.正弦(0)

// 导入子命名空间
从 编码 导入 JSON
JSON.编码(数据)
```

## 设计原则

1. **中文命名**：所有函数和类型使用中文命名，动词在前
2. **错误处理**：函数遇到错误时抛出中文命名的道语言异常，不使用错误码
3. **惰性求值**：处理集合和 I/O 时优先使用惰性求值（如 `逐行读取`）
4. **零依赖**：标准库仅依赖 Python 标准库，不引入第三方包

## 异常类型

标准库函数可能抛出以下异常：

| 异常类型 | 说明 | 典型场景 |
|---------|------|---------|
| `类型错误` | 参数类型不正确 | 传入非字符串到文本函数 |
| `运行时错误` | 运行时操作失败 | 文件不存在、正则表达式无效 |
| `索引错误` | 索引越界 | 空堆弹出、空队列弹出 |
| `断言失败` | 断言不成立 | 断言等于失败 |
