# 文本模块 API

模块路径：`文本`

## 导入方式

```
从 文本 导入 分割, 连接
从 文本 导入 正则匹配
导入 文本
```

## 函数列表

### 分割

按指定分隔符将字符串分割为列表。

**签名**：`分割(文本: 文本, 分隔符: 文本) -> 列表`

**参数**：
- `文本`：要分割的字符串
- `分隔符`：分隔符字符串，不能为空

**返回**：分割后的字符串列表

**异常**：
- `类型错误`：分隔符为空字符串，或参数类型不正确

**示例**：
```
定义 词语 = 分割("苹果,香蕉,橙子", ",")
打印(词语)  // ["苹果", "香蕉", "橙子"]
```

---

### 连接

用指定连接符将列表中的元素连接为一个字符串。

**签名**：`连接(列表: 列表, 连接符: 文本) -> 文本`

**参数**：
- `列表`：要连接的列表
- `连接符`：连接符字符串

**返回**：连接后的字符串

**异常**：
- `类型错误`：参数类型不正确

**示例**：
```
打印(连接(["苹果", "香蕉", "橙子"], " 和 "))  // "苹果 和 香蕉 和 橙子"
打印(连接([], ","))  // ""
```

---

### 替换

将字符串中的指定子串替换为新子串。

**签名**：`替换(文本: 文本, 旧串: 文本, 新串: 文本, 次数: 整数 = -1) -> 文本`

**参数**：
- `文本`：原始字符串
- `旧串`：要替换的子串
- `新串`：替换后的子串
- `次数`：最大替换次数，-1 表示全部替换

**返回**：替换后的字符串

**示例**：
```
打印(替换("你好世界", "世界", "道语言"))  // "你好道语言"
打印(替换("aaa", "a", "b", 次数=2))  // "bba"
```

---

### 包含

检查字符串是否包含指定子串。

**签名**：`包含(文本: 文本, 子串: 文本) -> 布尔`

**参数**：
- `文本`：原始字符串
- `子串`：要查找的子串

**返回**：包含返回 `真`，否则返回 `假`

**异常**：
- `类型错误`：参数类型不正确

**示例**：
```
打印(包含("你好世界", "世界"))  // 真
打印(包含("你好世界", "宇宙"))  // 假
```

---

### 去空格

去除字符串首尾的空白字符。

**签名**：`去空格(文本: 文本) -> 文本`

**参数**：
- `文本`：原始字符串

**返回**：去除首尾空白后的字符串

**示例**：
```
打印(去空格("  你好，世界！  "))  // "你好，世界！"
```

---

### 查找

查找子串在字符串中首次出现的位置。

**签名**：`查找(文本: 文本, 子串: 文本) -> 整数`

**参数**：
- `文本`：原始字符串
- `子串`：要查找的子串

**返回**：子串首次出现的位置索引，未找到返回 -1

**示例**：
```
打印(查找("你好世界", "世界"))  // 2
打印(查找("你好世界", "宇宙"))  // -1
```

---

### 截取

截取字符串的子串。

**签名**：`截取(文本: 文本, 开始: 整数, 结束: 整数 = 空) -> 文本`

**参数**：
- `文本`：原始字符串
- `开始`：起始位置（含）
- `结束`：结束位置（不含），省略则截取到末尾

**返回**：截取的子串

**示例**：
```
打印(截取("你好世界", 2, 4))  // "世界"
打印(截取("你好世界", 2))  // "世界"
```

---

### 大写

将字符串转换为大写。

**签名**：`大写(文本: 文本) -> 文本`

**参数**：
- `文本`：原始字符串

**返回**：大写字符串

**示例**：
```
打印(大写("hello"))  // "HELLO"
```

---

### 小写

将字符串转换为小写。

**签名**：`小写(文本: 文本) -> 文本`

**参数**：
- `文本`：原始字符串

**返回**：小写字符串

**示例**：
```
打印(小写("HELLO"))  // "hello"
```

---

### 正则匹配

使用正则表达式匹配字符串，返回所有匹配结果。

**签名**：`正则匹配(文本: 文本, 模式: 文本) -> 列表`

**参数**：
- `文本`：原始字符串
- `模式`：正则表达式模式

**返回**：所有匹配结果的列表，无匹配返回空列表

**异常**：
- `运行时错误`：正则表达式语法无效

**示例**：
```
打印(正则匹配("我的电话是13812345678", "[0-9]{11}"))  // ["13812345678"]
打印(正则匹配("无数字", "[0-9]+"))  // []
```

---

### 分割全部

按正则表达式分割字符串。

**签名**：`分割全部(文本: 文本, 模式: 文本) -> 列表`

**参数**：
- `文本`：原始字符串
- `模式`：正则表达式模式

**返回**：分割后的字符串列表

**异常**：
- `运行时错误`：正则表达式语法无效

**示例**：
```
打印(分割全部("苹果12香蕉34橙子", "[0-9]+"))  // ["苹果", "香蕉", "橙子"]
```

---

### 替换全部

按正则表达式替换所有匹配。

**签名**：`替换全部(文本: 文本, 模式: 文本, 替换: 文本) -> 文本`

**参数**：
- `文本`：原始字符串
- `模式`：正则表达式模式
- `替换`：替换字符串

**返回**：替换后的字符串

**异常**：
- `运行时错误`：正则表达式语法无效

**示例**：
```
打印(替换全部("a1b2c3", "[0-9]", "X"))  // "aXbXcX"
```
