# 测试模块 API

模块路径：`测试`

## 导入方式

```
从 测试 导入 测试套件, 断言等于, 断言为真
从 测试 导入 断言不等于, 断言为假, 断言为空, 断言包含, 断言抛出
从 测试 导入 测试报告
导入 测试
```

## 函数列表

### 测试套件

创建一个测试套件。

**签名**：`测试套件(名称: 文本) -> DaoTestSuite`

**参数**：
- `名称`：测试套件名称

**示例**：
```
定义 套件 = 测试套件("加法函数测试")
```

---

### 断言等于

断言两个值相等。

**签名**：`断言等于(期望: 任意, 实际: 任意) -> 空`

**异常**：`断言失败`：两个值不相等

**示例**：
```
断言等于(2 + 3, 5)
```

---

### 断言不等于

断言两个值不相等。

**签名**：`断言不等于(期望: 任意, 实际: 任意) -> 空`

**异常**：`断言失败`：两个值相等

---

### 断言为真

断言值为真。

**签名**：`断言为真(值: 任意) -> 空`

**异常**：`断言失败`：值为假

**示例**：
```
断言为真(包含("你好世界", "世界"))
```

---

### 断言为假

断言值为假。

**签名**：`断言为假(值: 任意) -> 空`

**异常**：`断言失败`：值为真

---

### 断言为空

断言值为空。

**签名**：`断言为空(值: 任意) -> 空`

**异常**：`断言失败`：值非空

**示例**：
```
断言为空("")
断言为空([])
```

---

### 断言包含

断言集合包含指定元素。

**签名**：`断言包含(集合: 任意, 元素: 任意) -> 空`

**异常**：`断言失败`：集合不包含元素

**示例**：
```
断言包含([1, 2, 3], 2)
断言包含("你好世界", "世界")
```

---

### 断言抛出

断言函数调用会抛出异常。

**签名**：`断言抛出(函数: 函数, 异常类型: 类型 = 空) -> 空`

**异常**：`断言失败`：函数未抛出异常

**示例**：
```
断言抛出(函数() => 读取("不存在的文件.txt"))
```

---

### 测试报告

生成测试套件的执行报告。

**签名**：`测试报告(套件: DaoTestSuite) -> 文本`

**异常**：`运行时错误`：参数不是测试套件类型

**示例**：
```
定义 套件 = 测试套件("示例")
// ... 执行测试 ...
打印(测试报告(套件))
```

---

## 自定义类型

### DaoTestSuite

测试套件，用于组织和记录测试结果。

**方法**：

| 方法名 | 签名 | 说明 |
|--------|------|------|
| `start` | `start() -> 空` | 开始计时 |
| `record_pass` | `record_pass(名称) -> 空` | 记录通过的测试 |
| `record_fail` | `record_fail(名称, 期望=空, 实际=空) -> 空` | 记录失败的测试 |
| `elapsed_ms` | `elapsed_ms() -> 小数` | 获取已用时间（毫秒） |
| `summary` | `summary() -> 文本` | 生成测试报告摘要 |

**属性**：

| 属性名 | 类型 | 说明 |
|--------|------|------|
| `name` | 文本 | 套件名称 |
| `passed` | 整数 | 通过的测试数 |
| `failed` | 整数 | 失败的测试数 |
