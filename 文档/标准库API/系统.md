# 系统模块 API

模块路径：`系统`

## 导入方式

```
从 系统 导入 取环境变量, 设环境变量
从 系统 导入 命令行参数, 执行命令
从 系统 导入 当前目录, 切换目录, 退出
从 系统 导入 平台
导入 系统
```

## 函数列表

### 取环境变量

获取环境变量的值。

**签名**：`取环境变量(名称: 文本, 默认值: 文本 = 空) -> 文本`

**参数**：
- `名称`：环境变量名
- `默认值`：变量不存在时的默认值，省略则抛出异常

**异常**：`运行时错误`：变量不存在且未提供默认值

**示例**：
```
定义 路径 = 取环境变量("PATH")
定义 值 = 取环境变量("MY_VAR", "默认值")
```

---

### 设环境变量

设置环境变量的值。

**签名**：`设环境变量(名称: 文本, 值: 文本) -> 空`

**示例**：
```
设环境变量("MY_VAR", "hello")
```

---

### 命令行参数

获取命令行参数列表。

**签名**：`命令行参数() -> 列表`

**返回**：命令行参数列表，第一个元素为程序路径

**示例**：
```
定义 参数 = 命令行参数()
打印(参数)  // ["main.py", "arg1", "arg2"]
```

---

### 执行命令

执行系统命令并返回输出。

**签名**：`执行命令(命令: 文本) -> 文本`

**参数**：
- `命令`：要执行的命令字符串

**返回**：命令的标准输出

**异常**：
- `运行时错误`：命令执行失败（非零退出码）或超时（默认60秒）

**示例**：
```
定义 输出 = 执行命令("echo hello")
打印(输出)  // "hello\n"
```

---

### 当前目录

获取当前工作目录。

**签名**：`当前目录() -> 文本`

**示例**：
```
打印(当前目录())  // "/home/user/project"
```

---

### 切换目录

切换当前工作目录。

**签名**：`切换目录(路径: 文本) -> 空`

**异常**：`运行时错误`：目录不存在

---

### 退出

退出程序。

**签名**：`退出(退出码: 整数 = 0) -> 空`

**参数**：
- `退出码`：退出码，0 表示正常退出

**示例**：
```
退出(0)  // 正常退出
退出(1)  // 异常退出
```

---

## 常量

| 常量名 | 类型 | 说明 |
|--------|------|------|
| `平台` | 文本 | 当前平台标识（如 `win32`、`linux`、`darwin`） |
