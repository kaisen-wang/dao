# 编码模块 API

模块路径：`编码`

## 导入方式

```
从 编码 导入 JSON, CSV
从 编码 导入 Base64, URL
导入 编码
```

编码模块采用子命名空间设计，每个子模块作为字典对象访问其内部函数。

---

## 子命名空间：JSON

### JSON.编码

将数据编码为 JSON 字符串。

**签名**：`JSON.编码(数据: 任意, 缩进: 整数 = 0) -> 文本`

**参数**：
- `数据`：要编码的数据（字典、列表等）
- `缩进`：缩进空格数，0 表示不缩进

**返回**：JSON 格式字符串

**异常**：`类型错误`：数据无法序列化为 JSON

**示例**：
```
定义 数据 = {"姓名": "张三", "年龄": 25}
打印(JSON.编码(数据))  // '{"姓名":"张三","年龄":25}'
打印(JSON.编码(数据, 缩进=4))  // 美化输出
```

---

### JSON.解码

将 JSON 字符串解码为数据对象。

**签名**：`JSON.解码(文本: 文本) -> 任意`

**异常**：
- `运行时错误`：JSON 格式无效
- `类型错误`：参数不是文本类型

**示例**：
```
定义 对象 = JSON.解码('{"姓名":"张三","年龄":25}')
打印(对象["姓名"])  // "张三"
```

---

## 子命名空间：CSV

### CSV.解析

将 CSV 文本解析为字典列表。

**签名**：`CSV.解析(文本: 文本) -> 列表`

**返回**：字典列表，每个字典的键为列名

**异常**：
- `运行时错误`：CSV 格式错误
- `类型错误`：参数不是文本类型

**示例**：
```
定义 表格 = CSV.解析("姓名,年龄\n张三,25\n李四,30")
// => [{"姓名": "张三", "年龄": "25"}, {"姓名": "李四", "年龄": "30"}]
```

---

### CSV.生成

将字典列表生成为 CSV 文本。

**签名**：`CSV.生成(数据: 列表) -> 文本`

**参数**：
- `数据`：字典列表，每个字典代表一行

**异常**：`运行时错误`：列表为空

**示例**：
```
定义 文本 = CSV.生成([{"姓名": "张三", "年龄": "25"}])
打印(文本)  // "姓名,年龄\n张三,25\n"
```

## 子命名空间：Base64

### Base64.编码

将字符串编码为 Base64 格式。

**签名**：`Base64.编码(文本: 文本) -> 文本`

**异常**：`类型错误`：参数不是文本类型

**示例**：
```
打印(Base64.编码("你好，世界！"))  // "5L2g5aW977yM5LiW55WM77yB"
```

---

### Base64.解码

将 Base64 字符串解码为原始字符串。

**签名**：`Base64.解码(文本: 文本) -> 文本`

**异常**：
- `运行时错误`：Base64 格式无效
- `类型错误`：参数不是文本类型

**示例**：
```
打印(Base64.解码("5L2g5aW977yM5LiW55WM77yB"))  // "你好，世界！"
```

---

## 子命名空间：URL

### URL.编码

对 URL 字符串进行编码。

**签名**：`URL.编码(文本: 文本) -> 文本`

**异常**：`类型错误`：参数不是文本类型

**示例**：
```
打印(URL.编码("你好 世界"))  // "%E4%BD%A0%E5%A5%BD%20%E4%B8%96%E7%95%8C"
```

---

### URL.解码

对 URL 编码字符串进行解码。

**签名**：`URL.解码(文本: 文本) -> 文本`

**异常**：`类型错误`：参数不是文本类型

**示例**：
```
打印(URL.解码("%E4%BD%A0%E5%A5%BD"))  // "你好"
```
