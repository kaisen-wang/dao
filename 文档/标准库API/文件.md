# 文件模块 API

模块路径：`文件`

## 导入方式

```
从 文件 导入 读取, 写入, 追加
从 文件 导入 存在, 列出目录, 逐行读取
从 文件 导入 复制, 移动, 删除, 创建目录, 文件信息
导入 文件
```

## 函数列表

### 读取

读取文件全部内容。

**签名**：`读取(路径: 文本) -> 文本`

**参数**：
- `路径`：文件路径

**返回**：文件内容字符串

**异常**：
- `运行时错误`：文件不存在或权限不足

**示例**：
```
定义 内容 = 读取("数据.txt")
```

---

### 写入

将内容写入文件（覆盖已有内容）。

**签名**：`写入(路径: 文本, 内容: 文本) -> 空`

**异常**：`运行时错误`：权限不足

**示例**：
```
写入("输出.txt", "你好，世界！")
```

---

### 追加

将内容追加到文件末尾。

**签名**：`追加(路径: 文本, 内容: 文本) -> 空`

**异常**：`运行时错误`：权限不足

**示例**：
```
追加("日志.txt", "新日志行\n")
```

---

### 存在

检查文件或目录是否存在。

**签名**：`存在(路径: 文本) -> 布尔`

**示例**：
```
如果 存在("配置.道")
    打印("配置文件已找到")
```

---

### 列出目录

列出目录下的所有文件和子目录。

**签名**：`列出目录(路径: 文本) -> 列表`

**异常**：
- `运行时错误`：目录不存在或路径不是目录

**示例**：
```
遍历 文件名 在 列出目录("./文档")
    打印(文件名)
```

---

### 逐行读取

惰性逐行读取文件，返回迭代器。适用于大文件处理，每次仅加载一行到内存。

**签名**：`逐行读取(路径: 文本) -> DaoLineIterator`

**示例**：
```
遍历 行 在 逐行读取("巨大的文件.txt")
    如果 包含(行, "错误")
        打印(行)
```

---

### 复制

复制文件。

**签名**：`复制(源: 文本, 目标: 文本) -> 空`

**异常**：
- `运行时错误`：源文件不存在或权限不足

---

### 移动

移动文件。

**签名**：`移动(源: 文本, 目标: 文本) -> 空`

**异常**：
- `运行时错误`：源文件不存在或权限不足

---

### 删除

删除文件。

**签名**：`删除(路径: 文本) -> 空`

**异常**：
- `运行时错误`：文件不存在或权限不足

---

### 创建目录

递归创建目录（含中间目录）。

**签名**：`创建目录(路径: 文本) -> 空`

**异常**：`运行时错误`：权限不足

**示例**：
```
创建目录("a/b/c")  // 递归创建所有中间目录
```

---

### 文件信息

获取文件元信息。

**签名**：`文件信息(路径: 文本) -> 字典`

**返回**：包含以下键的字典：

| 键 | 类型 | 说明 |
|------|------|------|
| `大小` | 整数 | 文件大小（字节） |
| `修改时间` | 小数 | 最后修改时间戳 |
| `是否目录` | 布尔 | 是否为目录 |
| `是否文件` | 布尔 | 是否为文件 |

**异常**：`运行时错误`：文件不存在

---

## 自定义类型

### DaoLineIterator

惰性文件行迭代器，由 `逐行读取` 函数返回。实现了迭代器协议，可在 `遍历` 语句中使用。文件读取完成后自动关闭文件句柄。
