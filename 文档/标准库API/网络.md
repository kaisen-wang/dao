# 网络模块 API

模块路径：`网络`

## 导入方式

```
从 网络 导入 请求, 创建服务器
从 网络 导入 URL编码, URL解码
导入 网络
```

## 函数列表

### 请求

发送 HTTP 请求。

**签名**：`请求(URL: 文本, 选项: 字典 = {}) -> DaoHttpResponse`

**参数**：
- `URL`：请求的 URL
- `选项`：可选配置字典，支持以下键：

| 键 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `方法` | 文本 | `"GET"` | HTTP 方法 |
| `请求头` | 字典 | `{}` | 请求头 |
| `数据` | 文本 | 空 | 请求体数据 |
| `超时` | 小数 | `30` | 超时秒数 |

**返回**：`DaoHttpResponse` 响应对象

**异常**：
- `运行时错误`：请求失败或超时

**示例**：
```
定义 响应 = 请求("https://api.example.com/data", {
    "方法": "GET",
    "请求头": {"Authorization": "Bearer xxx"}
})
打印(响应.状态码)  // 200
打印(响应.响应体)  // 响应内容
```

---

### 创建服务器

创建 HTTP 服务器。

**签名**：`创建服务器() -> DaoHttpServer`

**示例**：
```
定义 服务器 = 创建服务器()
服务器.路由("GET", "/你好", 函数(请求)
    返回 {"状态码": 200, "内容": "你好，世界！"}
)
服务器.启动(端口=8080)
```

---

### URL编码

对字符串进行 URL 编码。

**签名**：`URL编码(文本: 文本) -> 文本`

**异常**：`运行时错误`：参数不是文本类型

**示例**：
```
打印(URL编码("你好 世界"))  // "%E4%BD%A0%E5%A5%BD%20%E4%B8%96%E7%95%8C"
```

---

### URL解码

对 URL 编码字符串进行解码。

**签名**：`URL解码(文本: 文本) -> 文本`

**异常**：`运行时错误`：参数不是文本类型

**示例**：
```
打印(URL解码("%E4%BD%A0%E5%A5%BD"))  // "你好"
```

---

## 自定义类型

### DaoHttpResponse

HTTP 响应对象。

**属性**：

| 属性名 | 类型 | 说明 |
|--------|------|------|
| `状态码` | 整数 | HTTP 状态码 |
| `响应头` | 字典 | 响应头 |
| `响应体` | 文本 | 响应体内容 |

---

### DaoHttpServer

HTTP 服务器对象。

**方法**：

| 方法名 | 签名 | 说明 |
|--------|------|------|
| `路由` | `路由(方法: 文本, 路径: 文本, 处理函数: 函数) -> 空` | 注册路由处理函数 |
| `启动` | `启动(端口: 整数 = 8080) -> 空` | 启动服务器（阻塞） |

**异常**：
- `运行时错误`：端口被占用

**路由处理函数**：接收一个请求字典，返回一个包含 `状态码` 和 `内容` 的字典。

**请求字典**：

| 键 | 类型 | 说明 |
|------|------|------|
| `方法` | 文本 | 请求方法 |
| `路径` | 文本 | 请求路径 |
| `请求头` | 字典 | 请求头 |
| `请求体` | 文本 | 请求体 |
