# 3. 面向对象编程 (Object-Oriented Programming)

面向对象编程（OOP）通过封装、继承和多态来组织复杂的软件系统。"道"将提供一套直观的中文关键字来表达这些概念。

## 3.1 概念映射与关键字设计

| 概念 | 关键字 | 说明 |
|------|--------|------|
| 类 | `类型` | 定义一个类（选"类型"而非"类"，因为更贴合"一类事物"的中文表达） |
| 构造函数 | `初始化` | 对象创建时自动调用的方法 |
| 当前实例 | `本对象` | 等同于 `this` / `self` |
| 父类引用 | `父对象` | 调用父类方法 |
| 继承 | `继承自` | 声明继承关系 |
| 公有 | `公有`（默认） | 外部可访问 |
| 私有 | `私有` | 仅本类内部可访问 |
| 受保护 | `受保护` | 本类和子类可访问 |
| 静态 | `静态` | 属于类而非实例 |
| 抽象 | `抽象` | 必须由子类实现 |
| 接口/特征 | `特征` | 定义行为契约 |

## 3.2 基础：类、继承与多态

```
// 定义一个"动物"类型
类型 动物
    // 构造函数
    初始化(名字)
        本对象.名字 = 名字
        私有 本对象.年龄 = 0

    // 公有方法
    函数 吃(食物)
        打印(`{本对象.名字} 正在吃 {食物}`)

    函数 获取年龄()
        返回 本对象.年龄

    函数 设置年龄(年龄)
        如果 年龄 < 0
            抛出 错误("年龄不能为负数")
        本对象.年龄 = 年龄

// 定义一个"猫"类型，继承自"动物"
类型 猫 继承自 动物
    初始化(名字, 颜色)
        父对象.初始化(名字)
        本对象.颜色 = 颜色

    // 方法重写，实现多态
    函数 吃(食物)
        如果 食物 == "鱼"
            打印(`{本对象.名字} 开心地吃 {食物}`)
        否则
            打印(`{本对象.名字} 不想吃 {食物}`)

    // 猫特有的方法
    函数 喵()
        打印("喵～")

// 实例化对象
定义 小猫 = 猫("咪咪", "白色")
小猫.吃("鱼")      // 输出：咪咪 开心地吃 鱼
小猫.吃("骨头")    // 输出：咪咪 不想吃 骨头
小猫.喵()          // 输出：喵～
打印(小猫.名字)    // 输出：咪咪
// 打印(小猫.年龄) // 错误：无法访问私有属性
```

## 3.3 特征 (Trait / Interface)

特征定义了一组行为契约，类型可以实现多个特征，从而实现接口多态。特征解决了单继承的限制，提供了灵活的代码复用方式。

```
// 定义特征：可序列化
特征 可序列化
    函数 转为文本()  // 抽象方法，必须实现

// 定义特征：可比较
特征 可比较
    函数 比较(其他)  // 抽象方法

    // 特征可以提供默认实现
    函数 大于(其他)
        返回 本对象.比较(其他) > 0

    函数 小于(其他)
        返回 本对象.比较(其他) < 0

    函数 等于(其他)
        返回 本对象.比较(其他) == 0

// 定义特征：可打印
特征 可打印
    函数 显示()
        打印(本对象.转为文本())

// 一个类型可以继承一个父类 + 实现多个特征
类型 学生 继承自 人 实现 可序列化, 可比较, 可打印
    初始化(名字, 学号, 成绩)
        父对象.初始化(名字)
        本对象.学号 = 学号
        本对象.成绩 = 成绩

    // 实现 可序列化 的方法
    函数 转为文本()
        返回 `学生({本对象.名字}, {本对象.学号})`

    // 实现 可比较 的方法
    函数 比较(其他)
        返回 本对象.成绩 - 其他.成绩

// 特征可以用作参数类型约束（可选类型注解时）
函数 显示信息(对象: 可序列化)
    打印(对象.转为文本())
```

## 3.4 抽象类型

抽象类型不能直接实例化，必须由子类实现其抽象方法。它介于普通类型和特征之间：既可以有具体实现，也可以有抽象方法。

```
抽象 类型 图形
    // 抽象方法：子类必须实现
    抽象 函数 计算面积()
    抽象 函数 计算周长()

    // 具体方法：子类可以直接使用
    函数 描述()
        打印(`这是一个图形，面积为 {本对象.计算面积()}`)

类型 圆形 继承自 图形
    初始化(半径)
        本对象.半径 = 半径

    函数 计算面积()
        返回 3.14159 * 本对象.半径 ** 2

    函数 计算周长()
        返回 2 * 3.14159 * 本对象.半径

类型 矩形 继承自 图形
    初始化(宽, 高)
        本对象.宽 = 宽
        本对象.高 = 高

    函数 计算面积()
        返回 本对象.宽 * 本对象.高

    函数 计算周长()
        返回 2 * (本对象.宽 + 本对象.高)

// 定义 图形1 = 图形()  // 错误：抽象类型不能实例化
定义 圆 = 圆形(5)
圆.描述()  // 输出：这是一个图形，面积为 78.53975
```

## 3.5 静态方法与类方法

```
类型 数学工具
    // 静态方法：不需要实例，直接通过类型调用
    静态 函数 最大值(甲, 乙)
        返回 甲 如果 甲 > 乙 否则 乙

    静态 函数 最小值(甲, 乙)
        返回 甲 如果 甲 < 乙 否则 乙

// 直接通过类型名调用
打印(数学工具.最大值(3, 7))  // 7
打印(数学工具.最小值(3, 7))  // 3
```

## 3.6 运算符重载

允许自定义类型重新定义内置运算符的行为，使对象之间的操作更加自然。

```
类型 向量
    初始化(x, y)
        本对象.x = x
        本对象.y = y

    // 重载 + 运算符
    运算符 +(其他)
        返回 向量(本对象.x + 其他.x, 本对象.y + 其他.y)

    // 重载 * 运算符（标量乘法）
    运算符 *(标量)
        返回 向量(本对象.x * 标量, 本对象.y * 标量)

    // 重载 == 运算符
    运算符 ==(其他)
        返回 本对象.x == 其他.x 并且 本对象.y == 其他.y

    // 重载转为文本（用于打印）
    函数 转为文本()
        返回 `向量({本对象.x}, {本对象.y})`

定义 甲 = 向量(1, 2)
定义 乙 = 向量(3, 4)
定义 丙 = 甲 + 乙        // 向量(4, 6)
定义 丁 = 甲 * 3          // 向量(3, 6)
打印(丙)                   // 输出：向量(4, 6)
打印(甲 == 向量(1, 2))    // 输出：真
```

## 3.7 属性访问器

提供 `获取` 和 `设置` 语法糖，让属性访问可以附加逻辑，但使用时仍像普通属性一样自然。

```
类型 温度计
    初始化(摄氏度)
        私有 本对象._摄氏度 = 摄氏度

    // 属性获取器
    获取 摄氏度
        返回 本对象._摄氏度

    // 属性设置器
    设置 摄氏度(值)
        如果 值 < -273.15
            抛出 错误("温度不能低于绝对零度")
        本对象._摄氏度 = 值

    // 计算属性
    获取 华氏度
        返回 本对象._摄氏度 * 9 / 5 + 32

定义 温度 = 温度计(25)
打印(温度.摄氏度)   // 25（调用获取器）
打印(温度.华氏度)   // 77
温度.摄氏度 = 100   // 调用设置器
// 温度.摄氏度 = -300 // 错误：温度不能低于绝对零度
```
