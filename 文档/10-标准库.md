# 10. 标准库概要设计

"道"的标准库旨在提供日常开发所需的核心功能，所有 API 均使用中文命名，遵循统一的设计风格。

## 10.1 标准库模块总览

| 模块 | 说明 | 核心功能 |
|------|------|---------|
| `文本` | 字符串处理 | 分割、连接、查找、替换、正则 |
| `数学` | 数学计算 | 三角函数、随机数、统计 |
| `集合` | 数据结构 | 有序字典、双端队列、堆、树 |
| `文件` | 文件系统操作 | 读写、遍历、监听 |
| `网络` | 网络通信 | HTTP 请求/服务、WebSocket |
| `时间` | 日期与时间 | 解析、格式化、时区、计时 |
| `编码` | 数据编码 | JSON、XML、CSV、Base64 |
| `系统` | 系统交互 | 环境变量、命令行参数、进程 |
| `测试` | 测试框架 | 断言、测试套件、模拟 |
| `日志` | 日志记录 | 分级日志、格式化、输出 |

## 10.2 核心模块详细设计

### 10.2.1 文本模块

```
从 文本 导入 { 分割, 连接, 替换, 包含, 去空格, 正则匹配 }

定义 句子 = "  你好，世界！这是道语言。  "

// 去除首尾空白
打印(去空格(句子))  // "你好，世界！这是道语言。"

// 分割
定义 词语 = 分割("苹果,香蕉,橙子", ",")
打印(词语)  // ["苹果", "香蕉", "橙子"]

// 连接
打印(连接(词语, " 和 "))  // "苹果 和 香蕉 和 橙子"

// 替换
打印(替换("你好世界", "世界", "道语言"))  // "你好道语言"

// 包含检查
打印(包含("你好世界", "世界"))  // 真

// 正则表达式
定义 匹配 = 正则匹配("我的电话是13812345678", "[0-9]{11}")
打印(匹配)  // ["13812345678"]
```

### 10.2.2 文件模块

```
从 文件 导入 { 读取, 写入, 追加, 存在, 列出目录, 逐行读取 }

// 读取整个文件
定义 内容 = 读取("数据.txt")

// 写入文件
写入("输出.txt", "你好，世界！")

// 追加内容
追加("日志.txt", `[{时间.现在()}] 程序启动\n`)

// 检查文件是否存在
如果 存在("配置.道")
    打印("配置文件已找到")

// 遍历目录
遍历 文件名 在 列出目录("./文档")
    打印(文件名)

// 逐行读取大文件（惰性，节省内存）
遍历 行 在 逐行读取("巨大的文件.txt")
    如果 包含(行, "错误")
        打印(行)
```

### 10.2.3 网络模块

```
从 网络 导入 { 请求, 创建服务器 }

// === HTTP 客户端 ===
异步 函数 获取数据()
    定义 响应 = 等待 请求("https://api.example.com/data", {
        "方法": "GET",
        "请求头": {"Authorization": "Bearer xxx"}
    })

    如果 响应.状态码 == 200
        返回 响应.JSON()
    否则
        抛出 错误(`请求失败: {响应.状态码}`)

// === HTTP 服务器 ===
定义 服务器 = 创建服务器()

服务器.路由("GET", "/你好", 函数(请求)
    返回 {"状态码": 200, "内容": "你好，世界！"}
)

服务器.路由("POST", "/用户", 函数(请求)
    定义 数据 = 请求.JSON()
    // 处理创建用户逻辑...
    返回 {"状态码": 201, "内容": {"消息": "创建成功"}}
)

服务器.启动(端口=8080)
打印("服务器运行在 http://localhost:8080")
```

### 10.2.4 时间模块

```
从 时间 导入 { 现在, 解析, 格式化, 计时器 }

// 获取当前时间
定义 当前 = 现在()
打印(当前)  // 2026-02-09 15:30:00

// 格式化
打印(格式化(当前, "YYYY年MM月DD日"))  // 2026年02月09日

// 解析时间字符串

// 时间计算
定义 年龄 = 当前.年份 - 生日.年份
打印(`年龄: {年龄} 岁`)

// 计时器
定义 计时 = 计时器.开始()
// ... 执行一些操作 ...
打印(`耗时: {计时.停止()} 毫秒`)
```

### 10.2.5 编码模块

```
从 编码 导入 { JSON, CSV }

// JSON 操作
定义 数据 = {"姓名": "张三", "年龄": 25}
定义 文本 = JSON.编码(数据)          // '{"姓名":"张三","年龄":25}'
定义 对象 = JSON.解码(文本)          // 还原为字典

// JSON 美化输出
打印(JSON.编码(数据, 缩进=4))

// CSV 操作
定义 表格 = CSV.解析("姓名,年龄\n张三,25\n李四,30")
// => [{"姓名": "张三", "年龄": "25"}, {"姓名": "李四", "年龄": "30"}]
```

## 10.3 标准库设计原则

1. **命名一致性**：所有函数和类型使用中文命名，动词在前（如 `读取文件` 而非 `文件读取`）
2. **错误处理**：函数在遇到错误时抛出明确的异常，不使用错误码
3. **惰性优先**：处理集合和 I/O 时优先使用惰性求值，避免不必要的内存占用
4. **异步友好**：所有 I/O 操作同时提供同步和异步版本
5. **零依赖**：标准库不依赖第三方包，仅依赖虚拟机底层能力
