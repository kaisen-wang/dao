# 11. 关键字总表

本章列出"道"语言的所有保留关键字，按功能分类。这些关键字不能用作变量名、函数名或类型名。

## 11.1 基础关键字

| 关键字 | 用途 | 示例 |
|--------|------|------|
| `定义` | 声明可变变量 | `定义 x = 10` |
| `常量` | 声明不可变常量 | `常量 PI = 3.14` |
| `函数` | 定义函数 | `函数 加(a, b)` |
| `返回` | 从函数返回值 | `返回 结果` |
| `打印` | 输出到控制台 | `打印("你好")` |

## 11.2 控制流关键字

| 关键字 | 用途 | 示例 |
|--------|------|------|
| `如果` | 条件判断 | `如果 x > 0` |
| `否则如果` | 多分支条件 | `否则如果 x == 0` |
| `否则` | 默认分支 | `否则` |
| `当` | 当循环 (while) | `当 x < 10` |
| `遍历` | 遍历循环 (for) | `遍历 项 在 列表` |
| `在` | 遍历目标/成员检查 | `遍历 x 在 列表` |
| `从` | 范围起始 | `遍历 i 从 1 到 10` |
| `到` | 范围结束 | `遍历 i 从 1 到 10` |
| `步长` | 范围步长 | `从 0 到 10 步长 2` |
| `跳出` | 跳出循环 (break) | `跳出` |
| `继续` | 跳过本次循环 (continue) | `继续` |
| `匹配` | 模式匹配 | `匹配 值` |
| `情况` | 匹配分支 | `情况 0: ...` |
| `当` | 守卫条件 (在匹配中) | `情况 n 当 n > 0` |

## 11.3 面向对象关键字

| 关键字 | 用途 | 示例 |
|--------|------|------|
| `类型` | 定义类 | `类型 动物` |
| `初始化` | 构造函数 | `初始化(名字)` |
| `本对象` | 当前实例引用 | `本对象.名字` |
| `父对象` | 父类引用 | `父对象.初始化()` |
| `继承自` | 声明继承 | `类型 猫 继承自 动物` |
| `实现` | 实现特征 | `类型 A 实现 B, C` |
| `特征` | 定义接口/特征 | `特征 可序列化` |
| `抽象` | 修饰抽象类型/方法 | `抽象 类型 图形` |
| `公有` | 公有访问修饰 | 默认行为 |
| `私有` | 私有访问修饰 | `私有 本对象.x` |
| `受保护` | 受保护访问修饰 | `受保护 函数 辅助()` |
| `静态` | 静态方法/属性 | `静态 函数 创建()` |
| `运算符` | 运算符重载 | `运算符 +(其他)` |
| `获取` | 属性获取器 | `获取 名字` |
| `设置` | 属性设置器 | `设置 名字(值)` |

## 11.4 函数式编程关键字

| 关键字 | 用途 | 示例 |
|--------|------|------|
| `=>` | 匿名函数箭头 | `函数(x) => x * 2` |
| `产出` | 生成器/流产出值 | `产出 值` |
| `\|>` | 管道运算符 | `数据 \|> 映射(f)` |
| `...` | 展开/剩余运算符 | `[头, ...尾]` |

## 11.5 逻辑编程关键字

| 关键字 | 用途 | 示例 |
|--------|------|------|
| `逻辑` | 定义逻辑知识库 | `逻辑 家庭关系` |
| `事实` | 声明事实 | `事实: 父母("张三", "小明")` |
| `规则` | 声明规则 | `规则: 祖父母(...)` |
| `查询` | 执行逻辑查询 | `查询(知识库, ...)` |
| `?变量名` | 逻辑变量 | `?x`, `?谁` |
| `非` | 否定即失败 | `非 已封禁(?用户)` |
| `剪枝` | 阻止回溯 | `剪枝` |
| `添加事实` | 动态添加事实 | `添加事实(库, ...)` |
| `删除事实` | 动态删除事实 | `删除事实(库, ...)` |
