# 11. 关键字总表

本章列出"道"语言的所有保留关键字，按功能分类。这些关键字不能用作变量名、函数名或类型名。

> **注意**：部分关键字在特定上下文中允许作为标识符使用，详见各分类说明。关键字定义源码位于 `dao/tokens.py` 的 `KEYWORDS` 字典。

## 11.1 基础关键字

| 关键字 | 用途 | 示例 |
|--------|------|------|
| `定义` | 声明可变变量 | `定义 x = 10` |
| `常量` | 声明不可变常量 | `常量 PI = 3.14` |
| `设` | 变量赋值 | `设 x = 20` |
| `函数` | 定义函数 | `函数 加(a, b)` |
| `返回` | 从函数返回值 | `返回 结果` |

## 11.2 控制流关键字

| 关键字 | 用途 | 示例 |
|--------|------|------|
| `如果` | 条件判断 | `如果 x > 0` |
| `否则如果` | 多分支条件 | `否则如果 x == 0` |
| `否则` | 默认分支 | `否则` |
| `当` | 当循环 (while) | `当 x < 10` |
| `遍历` | 遍历循环 (for) | `遍历 项 在 列表` |
| `对于` | 遍历循环别名 | `对于 项 在 列表` |
| `在` | 遍历目标/成员检查 | `遍历 x 在 列表` |
| `从` | 范围起始/导入来源 | `遍历 i 从 1 到 10` |
| `到` | 范围结束 | `遍历 i 从 1 到 10` |
| `步长` | 范围步长 | `从 0 到 10 步长 2` |
| `跳出` | 跳出循环 (break) | `跳出` |
| `默认` | 匹配默认分支 | `默认: ...` |
| `继续` | 跳过本次循环 (continue) | `继续` |
| `匹配` | 模式匹配 | `匹配 值` |
| `情况` | 匹配分支 | `情况 0: ...` |
| `产出` | 生成器产出值 | `产出 值` |

> **说明**：`当` 在匹配守卫条件中也有使用（`情况 n 当 n > 0`），与循环 `当` 同名但上下文不同。`从` 同时用于模块导入（见 11.8）。`否则如果` 为组合关键字，词法分析器在遇到 `否则` 时会向前查看是否紧跟 `如果`，若是则合并为单个词元。

## 11.3 面向对象关键字

| 关键字 | 用途 | 示例 |
|--------|------|------|
| `类型` | 定义类 | `类型 动物` |
| `初始化` | 构造函数 | `初始化(名字)` |
| `本对象` | 当前实例引用 | `本对象.名字` |
| `父对象` | 父类引用 | `父对象.初始化()` |
| `继承自` | 声明继承 | `类型 猫 继承自 动物` |
| `实现` | 实现特征 | `类型 A 实现 B, C` |
| `特征` | 定义接口/特征 | `特征 可序列化` |
| `抽象` | 修饰抽象类型/方法 | `抽象 类型 图形` |
| `私有` | 私有访问修饰 | `私有 本对象.x` |
| `受保护` | 受保护访问修饰 | `受保护 函数 辅助()` |
| `静态` | 静态方法/属性 | `静态 函数 创建()` |
| `运算符` | 运算符重载 | `运算符 +(其他)` |
| `获取` | 属性获取器 | `获取 名字` |
| `设置` | 属性设置器 | `设置 名字(值)` |

> **说明**：公有访问为默认行为，无需关键字修饰。`初始化`、`本对象`、`父对象`、`获取`、`设置` 在特定上下文中允许作为标识符或成员名使用。

## 11.4 逻辑编程关键字

| 关键字 | 用途 | 示例 |
|--------|------|------|
| `逻辑` | 定义逻辑知识库 | `逻辑 家庭关系` |
| `事实` | 声明事实 | `事实: 父母("张三", "小明")` |
| `规则` | 声明规则 | `规则: 祖父母(...)` |
| `查询` | 执行逻辑查询 | `查询(知识库, ...)` |
| `非` | 否定即失败 | `非 已封禁(?用户)` |
| `剪枝` | 阻止回溯 | `剪枝` |
| `在范围` | 约束变量取值域 | `?x 在范围 1..10` |

> **说明**：`查询` 在特定上下文中允许作为标识符使用。逻辑变量以 `?` 前缀标识（如 `?x`、`?谁`），属于特殊语法而非关键字。`添加事实` 和 `删除事实` 为内置函数，非关键字。

## 11.5 元编程关键字

| 关键字 | 用途 | 示例 |
|--------|------|------|
| `定义宏` | 定义编译期宏 | `定义宏 除非(...)` |
| `引述` | 代码转 AST | `引述 { ... }` |
| `引用` | 代码转 AST（同义词） | `引用 { ... }` |
| `注入` | AST 插入 | `注入 表达式` |

> **说明**：`引用` 与 `引述` 功能相同，为同义关键字。宏调用前缀 `!` 和注入前缀 `$` 为运算符，非关键字。

## 11.7 并发编程关键字

| 关键字 | 用途 | 示例 |
|--------|------|------|
| `异步` | 声明异步函数 | `异步 函数 获取()` |
| `等待` | 等待异步结果 | `等待 获取数据()` |
| `全部` | 并发等待所有 | `全部 { ... }` |
| `竞速` | 竞速等待 | `竞速 { ... }` |
| `并行` | 启动协程 | `并行 { ... }` |
| `通道` | 创建通道 | `定义 ch = 通道()` |
| `发送` | 发送数据到通道 | `发送(ch, 数据)` |
| `接收` | 从通道接收数据 | `接收(ch)` |
| `选择` | 多通道选择 | `选择 { ... }` |
| `超时` | 超时控制 | `超时 1000` |
| `互斥锁` | 互斥锁 | `定义 锁 = 互斥锁()` |
| `原子整数` | 原子整数类型 | `定义 n = 原子整数(0)` |
| `原子布尔` | 原子布尔类型 | `定义 b = 原子布尔(假)` |
| `同步` | 同步代码块 | `同步 锁 { ... }` |

> **说明**：本分类关键字在特定上下文中允许作为标识符使用。`运行异步` 为内置函数名，映射为标识符而非关键字。

## 11.8 模块系统关键字

## 11.12 运算符与特殊语法

以下运算符和特殊语法不属于关键字，但在语言中具有特殊含义：

| 符号 | 用途 | 示例 |
|------|------|------|
| `=>` | 匿名函数箭头 | `函数(x) => x * 2` |
| `\|>` | 管道运算符 | `数据 \|> 映射(f)` |
| `...` | 展开/剩余运算符 | `[头, ...尾]` |
| `..` | 整数闭区间 | `1..10` |
| `!` | 宏调用前缀 | `!计时({...})` |
| `$` | 宏注入前缀 | `$表达式` |
| `?` | 逻辑变量前缀 | `?x`, `?谁` |
