## 1. 文件命名不一致

**问题**：并发解析器相关文件命名混乱

| 预期文件名 | 实际文件名 | 说明 |
|-----------|-----------|------|
| `concurrency_parser.py` | 不存在 | 文档中列出的 `currency_parser.py` 实际也不存在 |
| `concurrency.py` | `currency.py` | 拼写错误（currency ≠ concurrency），文件内容为 `ConcurrencyParser` |

**检查结果**：
- `dao/parser/modules/currency_parser.py` **不存在**，无需重命名
- `dao/parser/modules/currency.py` **存在**，文件内类名为 `ConcurrencyParser`，确认是文件名拼写错误
- `__init__.py` 中 `from .currency import ConcurrencyParser` 需同步更新

**任务**：
- [ ] 将 `dao/parser/modules/currency.py` 重命名为 `concurrency.py`
- [ ] 更新 `dao/parser/modules/__init__.py` 中的 import 语句：`from .currency` → `from .concurrency`
- [ ] 检查并更新测试文件中的 import 语句（当前测试通过 `modules` 包间接引用，可能无需修改）
