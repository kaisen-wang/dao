# 综合项目：回合制策略游戏（卡牌对战/杀戮尖塔风格）

覆盖道语言全部功能模块的综合实战项目。
道语言文档：/home/carson/code/test/道/文档

## 项目结构

```
战棋/
├── 入口.道                    # 入口
├── 核心/
│   ├── 引擎.道                # 游戏主循环、状态管理
│   ├── 事件.道                # 事件总线（异步消息通道）
│   └── 日志.道                # 日志系统（标准库 log）
├── 实体/
│   ├── 角色.道                # 类型：角色基类
│   ├── 卡牌.道                # 枚举：卡牌类型、稀有度
│   ├── 技能.道                # 抽象类型：技能基类 + 具体技能
│   ├── 装备.道                # 特征：可装备接口
│   └── 状态效果.道            # 枚举 + 特征：Buff/Debuff
├── 战斗/
│   ├── 计算器.道              # 管道链：攻击计算 |> 元素加成 |> 暴击 |> 减伤
│   ├── 行为树.道              # 逻辑编程：AI 决策规则
│   └── 卡牌效果.道            # 模式匹配：匹配 卡牌.类型 => 效果逻辑
├── DSL/
│   ├── 技能定义.道            # 宏：!技能("火球") => 引述 { 定义宏 ... }
│   ├── 事件脚本.道            # 宏：!事件("抽牌") => 模式匹配宏展开
│   └── 条件模板.道            # 宏：!条件(血量 < 30%) => AST 变换
├── 并发/
│   └── 网络对战.道            # 异步/通道/选择器：回合同步、超时处理
├── 工具/
│   ├── 序列化.道              # 标准库 file + encoding：存档读写
│   ├── 测试框架.道            # 标准库 test：单元测试
│   └── 类型定义.道            # 类型别名：联合类型、函数类型注解
└── examples/                  # 可运行示例 .道 文件
    ├── 试玩对战.道
    └── AI演示.道
```

## 覆盖检测

运行以下命令检验所有模块是否被项目覆盖：

```bash
# 统计项目用到的道语言关键字
cd 源码
python -c "
from dao.lexer import Lexer
from dao.tokens import TokenType
keys = set()
for f in glob.glob('../战棋/**/*.道', recursive=True):
    with open(f) as fh:
        tokens = Lexer(fh.read(), f).tokenize()
        for t in tokens:
            keys.add(t.type.name)
print(f'覆盖关键字: {len(keys)}')
print(sorted(keys))
"
```
