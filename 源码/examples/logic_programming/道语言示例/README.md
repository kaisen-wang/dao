# 道语言逻辑编程示例

## 概述

这个目录展示了道语言的逻辑编程功能。逻辑编程是一种声明式编程范式，允许开发者通过声明事实和规则来描述问题，由语言引擎自动进行推理和搜索答案。

## 核心概念

### 知识库
知识库是存储事实和规则的容器：
```python
kb = KnowledgeBase("家庭关系")
kb.add_fact(LogicStruct("父母", ("张三", "小明")))
kb.add_rule(rule_head, rule_body)
```

### 事实
事实是已知的基本关系：
```
事实: 父母("张三", "小明")
事实: 父母("李四", "小明")
```

### 规则
规则定义了如何从已知事实推导出新事实：
```
规则: 祖父母(?祖, ?孙) 如果 父母(?祖祖, ?父) 并且 父母(?父, ?孙)
```

### 查询
查询用于从知识库中检索信息：
```python
goal = LogicStruct("祖父母", ("张三", "?孙))
results = solve(kb, goal)
```

## 推理能力

道语言的逻辑编程引擎支持：

1. **直接事实查询**：从知识库中检索已知关系
2. **规则推理**：通过规则自动推导新关系
3. **递归查询**：支持递归规则的推理
4. **多解返回**：可以返回所有可能的解
5. **变量绑定**：自动处理变量绑定和替换

## 技术实现

道语言的逻辑编程引擎基于以下技术：

- **统一化算法**：Martelli-Montanari算法
- **回溯搜索**：深度优先搜索（DFS）
- **Trail Stack**：支持完整的回溯机制
- **约束求解**：支持范围约束和全不同约束
- **类型推断**：自动将参数转换为LogicAtom或LogicVariable
